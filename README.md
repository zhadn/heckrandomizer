# Heckrandomizer
Heckrandomizer is a Chrono Cross randomizer.

The Heckrandomizer is currently in development and not playable.

## About the Heckrandomizer
The 1.0 release of the Heckrandomizer is not scoped for the full game.

Once the 1.0 is released, and stable, other content, including playing as Lynx (post Fort Dragonia), will be planned according to community interest.

**This project is a work in progress.**

### Open World modifications
The following are major changes that allow the Heckrandomizer to be playable as an open randomizer.
- Skips the dream sequence.
- Starts in Serge's room.
- Skips all cutscenes.

With additional modifications to Serge for open world accessibility:
- Serge starts with the Tele-Porter key item (allows setting party members).
- Serge starts with the Astral Amulet key item (allows traveling between worlds).
- Serge starts with Korcha's boat (allows ocean travel).

With additional modifications to Key Items to improve the open world randomizer experience.
- Ice Breath is required (no longer optional) to enter Mt. Pyre.

### Character modifications
The following are major changes that allow missable characters to be playable in the Heckrandomizer.

- Kid starts with Serge or becomes unlockable.
- Leena starts with Serge or becomes unlockable.
- Mojo is permanently unlockable.
- Guile path should remain open.
- Nikki path should remain open.
- Pierre path should remain open.
- Korcha is permanently unlockable.
- Mel is permanently unlockable.
- Razzly is permanently unlockable.
- Macha is permanently unlockable.
- Glenn is permanently unlockable.
- Doc is permanently unlockable.
- Pip starts unlockable on the S.S. Invincible.  

### Battle modifications
- Characters start with max level element grid. 

## Win condition
- Defeat Lynx in Fort Dragonia.
