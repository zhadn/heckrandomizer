# Modding Chrono Cross
A short guide on modding Chrono Cross: The Radical Dreamers Edition.

## Game files
In Windows, the default directory is `\Steam\steamapps\common`.

## Modifying .DAT files
`.dat` files can be converted to `.zip` files. Extract files to edit directly.

There are nine `.dat` files:
- `cdrom.dat`
- `cdrom2.dat`
- `font.dat`
- `hd.dat`
- `lang.dat`
- `launcher.dat`
- `movie.dat`
- `radicaldreamers.dat`
- `ui.dat`
